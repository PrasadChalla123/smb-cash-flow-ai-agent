# SMB Cash Flow Risk Predictor AI Agent

## Overview
The SMB Cash Flow Risk Predictor is an AI-powered financial forecasting system that helps Small & Medium Businesses (SMBs) predict upcoming cash-flow risks.  
It analyzes revenues, expenses, receivables, and payables to forecast future cash positions and classify months as Safe, Warning, or Critical — with human-readable AI insights.

---

## Features

- Upload your company’s monthly income & expense CSV  
- Choose forecast duration dynamically (e.g., next 3–12 months)  
- Automatic forecasting using Facebook Prophet  
- Risk classification using rule-based logic  
- AI-generated summary using OpenAI GPT  
- Interactive web UI built with Streamlit  
- Modular architecture with Flask backend API
